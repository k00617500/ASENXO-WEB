## Overview ğŸ§©

**ASENXO Web** is a centralized, browser-based system designed to modernize the **DOST-SETUP 4.0** project lifecycle.  
It unifies application processing, procurement tracking, compliance reporting, GIS visualization, and predictive analytics into one platform.

<div align="center">

| Module | Description |
|------|-------------|
| Application Portal | Digital submission and evaluation of MSME proposals |
| GIS Mapping | TNA Geographic visualization of MSME beneficiaries |
| Procure-to-Pay | End-to-end tracking from approval to liquidation |
| Predictive Analytics | Regression-based forecasting of risks and delays |
| Workflow Engine | Rule-based validation and task routing |
| Admin Dashboard | Real-time project and fund monitoring |
